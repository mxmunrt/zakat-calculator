# Project Context — ZakatCalc

## Overview
A Zakat calculator web app that helps Muslims determine their Zakat obligations. Multi-step form with live metal prices, deployed at **zakatcalc.mrtalukdar.dev**.

## Tech Stack
- **Frontend:** HTML, vanilla CSS, JavaScript (ES Modules)
- **Build:** Vite 7
- **Hosting:** Vercel (auto-deploys on push to `main`)
- **API:** Vercel Serverless Function (`api/prices.js`)

## Architecture

### Live Metal Prices
- `api/prices.js` — serverless function that proxies requests to the **Swissquote forex feed** (no API key needed)
- Fetches both gold (XAU) and silver (XAG) prices per troy ounce, converts to per-gram
- Server-side cache: **24 hours** (`s-maxage=86400`)
- Client-side cache: **24 hours** (in-memory)
- Supports GBP, USD, EUR
- API URL pattern: `https://forex-data-feed.swissquote.com/public-quotes/bboquotes/instrument/{METAL}/{CURRENCY}`

### Nisab Calculation
- Uses **silver standard**: 612.36g × live silver price per gram
- Displayed in the Nisab banner with per-gram price shown, e.g. "Based on 612.36g of silver at today's price (£2.12/g)"
- Falls back to hardcoded estimates if the API fails

### Local Development
- `vite.config.js` contains a custom plugin (`devApiProxy`) that mimics the Vercel serverless function locally
- This means `/api/prices` works in both dev and production without any code changes

## File Structure
- `index.html` — all sections (hero, calculator, learn, footer)
- `src/main.js` — app entry, event wiring, price loading
- `src/calculator.js` — Zakat engine, price fetching, currency formatting
- `src/steps.js` — multi-step form controller
- `src/results.js` — results display with animated counter
- `src/style.css` — full design system (dark theme, emerald accents)
- `api/prices.js` — Vercel serverless price proxy

## Deployment
- **Repo:** github.com/mxmunrt/zakat-calculator
- **Domain:** zakatcalc.mrtalukdar.dev (CNAME → cname.vercel-dns.com)
- **Auto-deploy:** every `git push` to `main` triggers Vercel build

## Design Decisions
- Dark theme with emerald green (`#10b981`) accents and glassmorphism
- Fonts: Inter (UI) + Amiri (Arabic text)
- Byline "BY MRTALUKDAR.DEV" under hero title — subtle, muted, uppercase
- Islamic references based on Sheikh Majed Mahmoud's Fiqh of Zakat series
- Calculator uses 5 steps: Eligibility → Cash → Gold/Silver → Investments → Liabilities
- Zakat rate: 2.5% of net zakatable wealth above Nisab
